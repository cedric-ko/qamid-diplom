# Отчёт по итогам автоматизации

##  Было запланировано и фактически реализовано:
* составление тестовых сценариев - составлено 39 тест-кейсов; 
* тестирование UI страниц "Путешествие дня" и "Оплата по карте", в том числе навигация и заполнение форм, взаимодействия приложения с БД - реализовано;
* составление баг-репортов по обнаруженным дефектам - 19 репортов;
* автоматизация тесторования - выполнено;
* составление тестовой документации - выполнено.

В процессе тестирования было принято решение отказаться от тестирования страницы "Кредит по данным карты", API-тестирования и параллельного запуска контейнера PostgreSQL в виду дефицита времени.

## Сработавшие риски
Отсутствие документации на тестируемое приложение и необходимость написания кода автотестов существенно увеличили временные затраты. В ходе работы с кодом страницы также было установлено отсутствие тестовых меток.

## Общий итог по времени
Планировалось временных затрат: 106 часов

Фактически затрачено: 51 час.

Обоснование расхождение: отказ от части запланированных работ, отсутствие релевантного опыта.
